# Assignment 4: Student Record Management System

## Description
A menu-driven C program that manages student records. This system provides comprehensive functionality to handle student information including adding, displaying, searching, updating, and deleting student records.

## Features
- **Add Student** - Add new student records with roll number, name, and marks
- **Display Students** - View all stored student records
- **Search Student** - Search for a specific student by roll number
- **Update Student** - Modify existing student information
- **Delete Student** - Remove student records from the system
- **Exit** - Safely close the program

## Files
- `assignment4.c` - Main C source code for the assignment

## How to Compile
```bash
gcc assignment4.c -o assignment4
```

## How to Run
```bash
./assignment4
```

## Requirements
- GCC compiler
- C standard library

## Data Structure
The program uses a structure to store:
- Roll Number
- Student Name
- Marks

## Concepts Covered
- Structure (struct) usage
- Dynamic data management
- CRUD operations (Create, Read, Update, Delete)
- Menu-driven programming
- User input validation
