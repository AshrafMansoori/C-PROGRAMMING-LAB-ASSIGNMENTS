# Assignment 3: Pointer Playground Program

## Description
A comprehensive C program that demonstrates pointer concepts and operations. This program showcases various pointer applications including memory addressing, dereferencing, swapping, and array traversal using pointers.

## Features
- **Memory Address** - Display memory addresses and pointer storage
- **Dereferencing Pointer** - Access values through pointer dereference
- **Swapping Two Numbers** - Swap integers using pointers
- **Array Traversal** - Traverse arrays using pointer arithmetic

## Files
- `assignment3.c` - Main C source code for the assignment

## How to Compile
```bash
gcc assignment3.c -o assignment3
```

## How to Run
```bash
./assignment3
```

## Requirements
- GCC compiler
- C standard library

## Concepts Covered
- Pointer declarations and initialization
- Address-of operator (&)
- Dereference operator (*)
- Pointer arithmetic
- Pass-by-reference using pointers
